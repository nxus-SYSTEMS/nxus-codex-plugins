# Contributing

Thanks for helping improve nxus.SYSTEMS Codex Plugins.

## Scope

This repository publishes Codex Plugin packages and documentation for nxus.SYSTEMS products. Keep contributions focused on shipped plugin behavior, install metadata, documentation clarity, examples referenced by the plugin, and reproducible validation.

## Public-Safety Rules

- Do not commit secrets, credentials, tokens, license keys, private paths, machine-specific paths, or personal environment details.
- Do not include unpublished product plans, private notes, or coordination material.
- Do not add MCP servers, app connectors, hooks, or runtime services to the nxusKit Codex Plugin package.
- Keep generated or temporary files out of the repository.

## Validation

Before opening a pull request, run the validation commands documented by the package you changed. For nxusKit Celerat, inspect `plugins/nxuskit/README.md` and the skill reference docs for expected behavior.
