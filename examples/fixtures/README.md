# Starter Fixtures

These fixtures are intentionally small. They give Codex a real project shape to inspect before applying one of the task recipes.

They do not include generated artifacts, embedded repositories, machine-specific SDK paths, or credentials.

## Fixtures

- `starters/rust-cli/` — empty Rust CLI.
- `starters/go-service/` — minimal Go command with provider flags.
- `starters/python-app/` — minimal Python package.
- `starters/cli-bash-workspace/` — shell workspace for JSON-first prototypes.
- `starters/direct-openai-python/` — direct OpenAI SDK usage for migration practice.
- `starters/local-provider-python/` — local-provider preference app.
